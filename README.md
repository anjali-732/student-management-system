# Student Management System (SMS)

A lightweight, error-free, and responsive **Student Management System** built with HTML5, CSS3, and Vanilla JavaScript. This application allows users to manage student records locally within the browser.

## 🚀 Features

* **Create**: Add new students with their Name, ID, and Email.
* **Read**: View all registered students in a clean, formatted table.
* **Delete**: Remove student records instantly.
* **Persistent Storage**: Data is saved to `localStorage`, so records remain even after refreshing the page or closing the browser.
* **Responsive Design**: Modern UI that works across different screen sizes.

## 🛠️ Technologies Used

* **HTML5**: Semantic structure of the application.
* **CSS3**: Custom styling using CSS variables and Grid/Flexbox for layout.
* **JavaScript (ES6)**: Logic for CRUD operations and Local Storage integration.
